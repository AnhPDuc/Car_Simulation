Set up environment

Step 1: Install Node.js (download at https://nodejs.org/en/download)

Step 2: Install three.js (using a terminal in your project folder)
# three.js
npm install --save three
# vite
npm install --save-dev vite

Step 3: Open DoHoaMayTinh.html and try it


How to use ?

-Use the arrow keys to move the car.
-Use the slider on the top left corner to adjust the brightness for the city
-Use the left mouse button to change the view of the city
-Use the right mouse button to change the location of the city
-Use the middle mouse to zoom in and out
